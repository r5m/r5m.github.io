    Добро пожаловать на сайт R5M Finance

    R5M Finance - это приложение для учета Ваших личных финансов:

    • Любые устройства и браузеры - Anroid 4.0.3+, IOS, Ubuntu Touch, Firefox OS и др.
    • Оптимизированный интерфейс для смартфонов и планшетов
    • Никаких сложных настроек: начните учет финансов в несколько кликов.
    • Не требуется регистрация аккаунта
    • Работает оффлайн
    • Неограниченное количество счетов
    • Поддержка более 100 валют
    • Резервное копирование и синхронизация данных через Ваш аккаунт в Dropbox
    • Русский или Английский язык интерфейса

    Установка

    На нашем сайте доступны три версии приложения для различных устройств. Все версии обладают одинаковой функциональностью:

    • Apk - предпочтительный вариант для устройств на базе Android с поддержкой стандартных элементов управления.
    • Firefox/B2G - подходит для всех устройств (Frefox OS, Windows, GNU/Linux, Android...) с установленным браузером Mozilla Firefox. Ссылка только для Firefox.
    • Онлайн-версия для других устройств/браузеров. Не требует установки. Просто добавьте в закладки браузера.
